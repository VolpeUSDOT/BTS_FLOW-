Flow - API

This OpenAPI specification document describes the details of the API for data collected under the Freight Logistics Optimization Works (FLOW) program. This data is managed under the stewardship of the USDOT Bureau of Transportation Statistics.
More information: https://www.bts.gov/flow
Contact Info: btsdataportal@dot.gov
