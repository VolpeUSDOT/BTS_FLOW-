Flow - API

This OpenAPI specification document describes the details of the API for data collected under the Freight Logistics Optimization Works (FLOW) program. This data is managed under the stewardship of the USDOT Bureau of Transportation Statistics.
More information: https://www.bts.gov/flow
Contact Info: btsdataportal@dot.gov
Version: 0.1.0
BasePath: https://www.c3rs.bts.gov/flow-api/

get /api/v0/port/{portLocode}
List possible demand and supply signals at port (getPortInformation)

Path parameters

portLocode (required)
Path Parameter — UN/LOCODE for port location

Return type

PortInformation

Example data

Content-Type: application/json
{
  "portLocode" : "USSAV",
  "supplySignals" : [ "CHA20", "CHA20" ],
  "demandSignals" : [ "OC", "OC" ],
  "name" : "Port of Savannah, GA"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

OK PortInformation

404

Port identifier not found

get /api/v0/port/{portLocode}/daily
Retrieve daily data (portDailyData)
Retrieve daily data on demand and signal signals of ports.

Path parameters

portLocode (required)
Path Parameter — UN/LOCODE for port location

Query parameters

supplySignal (required)
Query Parameter — Identifier for supply signal (e.g. MTO for Marine Terminal Slots, CHA20 for Chassis-20ft, CHA40 for Chassis-40ft).
demandSignal (required)
Query Parameter — Identifier for demand signal (e.g OC for Ocean Carrier Bookings).
startDate (optional)
Query Parameter — start-date of window to retrieve information on.
endDate (optional)
Query Parameter — end-date of window to retrieve information on.

Return type

PortDataInformationPort

Example data

Content-Type: application/json
{
  "port" : "USSAV",
  "supplySignal" : "MTO",
  "days" : [ {
    "dateOccurred" : "2023-12-31",
    "supply" : [ {
      "dimension" : "TEU",
      "value" : 3346
    }, {
      "dimension" : "TEU",
      "value" : 3346
    } ],
    "demand" : [ null, null ]
  }, {
    "dateOccurred" : "2023-12-31",
    "supply" : [ {
      "dimension" : "TEU",
      "value" : 3346
    }, {
      "dimension" : "TEU",
      "value" : 3346
    } ],
    "demand" : [ null, null ]
  } ],
  "demandSignal" : "OC"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

OK PortDataInformationPort

403

Invalid authorization

404

Port and/or demand signal and/or supply signal not found

405

Invalid request parameter.

429

Request limit reached
